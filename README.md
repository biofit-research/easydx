# 알기쉬운 진단명 가이드

> 진단서 속 그 코드, 누구나 이해할 수 있게.

외래에서 가장 많이 등장하는 100가지 진단명(ICD-10 / KCD-8 코드)을, 의료진이 검수한 쉬운 한국어로 즉시 풀어드리는 웹 앱입니다.

🔗 **데모**: [GitHub Pages 링크 — 배포 후 추가]

---

## 만든 이유

진단서·검사결과지·처방전에 적힌 `J45`, `E11`, `I10` 같은 코드는 환자가 직접 읽기 어렵습니다. 인터넷을 검색해도 의학용어가 다른 의학용어로 설명되는 경우가 많아 의료 정보 격차가 그대로 남습니다.

이 앱은 환자·일반인을 위해 다음을 한 화면에서 제공합니다.

- 정식 한국어 병명과 영문명
- 고등학생 수준의 쉬운 풀이 + 일상 비유
- 환자가 실제로 느끼는 흔한 증상
- 병원에 바로 가야 할 응급 신호
- 함께 자주 진단되는 관련 코드

---

## 주요 기능

- **통합 검색** — 코드(J45), 한국어 병명(천식), 별칭(고혈압) 모두 매칭되는 자동완성
- **분야별 보기** — 18개 진료 분야 탭으로 필터링
- **응급 키워드 감지** — '응급', '의식', '호흡곤란' 등 입력 시 119 안내 우선 노출
- **신뢰도 라벨** — 각 풀이가 의료진 검수를 거쳤음을 명시
- **면책 고지 시스템** — 본문 하단 + 모달 + 푸터 3중 노출
- **출처 명시** — KCD-8, WHO ICD-10, HIRA 2024 통계 출처 표기
- **광고·제휴 없음** — 특정 병의원·의약품 추천 일체 없음

## 데이터 처리 구조 (3단 캐시)

API 호출을 최소화하면서도 모든 코드를 다룰 수 있는 3단 구조로 설계되어 있습니다.

| 단계 | 범위 | 처리 방식 |
|---|---|---|
| **Tier 1** — 내장 사전 | 외래 다빈도 100개 | 본 저장소에 JSON으로 내장, 의료진 사전 검수, 즉시 응답, API 비용 0원 |
| **Tier 2** — 서버 캐시 | 그 외 검색되는 코드 | 첫 사용자만 Claude API 호출 → 결과를 DB에 저장 → 이후 재사용 |
| **Tier 3** — 실시간 호출 | 희귀·신규 코드 | Claude API 직접 호출, 'AI 자동 생성' 라벨 명시 |

현재 공개 데모 버전에는 **Tier 1만** 구현되어 있습니다.

## 시작하기

### 가장 간단한 방법

1. 이 저장소를 다운로드 또는 클론
2. `index.html`을 더블클릭

별도 빌드·서버 없이 브라우저에서 즉시 작동합니다. React와 Babel은 CDN으로 로드됩니다.

```bash
git clone https://github.com/<사용자명>/<저장소명>.git
cd <저장소명>
open index.html   # macOS
start index.html  # Windows
```

### GitHub Pages로 배포

1. 저장소의 **Settings** → **Pages** 메뉴로 이동
2. **Source**를 `Deploy from a branch`로 설정
3. **Branch**를 `main` / `(root)`으로 선택 후 **Save**
4. 1~2분 후 `https://<사용자명>.github.io/<저장소명>/` 주소에서 접속 가능

### Vercel·Netlify로 배포

GitHub 저장소를 Vercel(vercel.com) 또는 Netlify(netlify.com)에 연결하면 클릭 몇 번으로 배포됩니다. 빌드 설정 없이 정적 파일 그대로 호스팅됩니다.

## 콘텐츠 편집 워크플로우

진단명 풀이 내용은 다음 4단계로 관리합니다.

```
[1] Claude로 초안 생성  →  [2] 엑셀에서 검수·편집  →  [3] JSON 변환  →  [4] 앱에 반영
   (개발자, 1회)              (의료진, 행 단위)         (개발자, 자동)    (개발자, 자동)
```

엑셀 템플릿(`ICD10_콘텐츠_편집_템플릿.xlsx`)은 의료진이 코드를 한 줄도 보지 않고 워드 편집과 동일한 방식으로 검수할 수 있도록 설계되었습니다. 자세한 작성 가이드는 템플릿 내 `작성_가이드` 시트를 참고하세요.

## 면책 고지

본 앱이 제공하는 모든 정보는 일반적인 건강 정보 제공을 위한 **참고 자료**이며, 의학적 진단·치료를 대체할 수 없습니다.

- 증상이 있거나 진단명에 대해 정확히 알고 싶다면 반드시 **의료진과 직접 상담**하시기 바랍니다.
- 응급 상황(흉통·호흡곤란·의식 저하·심한 출혈 등)에서는 검색이 아닌 **119 신고 또는 가까운 응급실 방문**을 우선해 주세요.
- 본 앱은 특정 의료기관·의약품을 추천하거나 광고하지 않습니다.
