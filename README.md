# 알기쉬운 진단명 가이드

> 진단서 속 그 코드, 실무자가 빠르게 이해할 수 있게.

외래에서 자주 마주치는 진단명(ICD-10 / KCD-8 코드)을, 의료진이 검수한 쉬운 한국어로 즉시 풀어주는 **실무용 참고 도구**입니다. 환자 자가진단용이 아니라, 코드를 다루는 현장 실무자가 "이 코드가 대략 무슨 뜻인지"를 빠르게 파악하도록 돕는 데 목적이 있습니다.

🔗 **데모**: https://easy-diagnosis-guide.vercel.app/

---

## ⚠️ 먼저 읽어주세요 — 이 도구의 성격

이 앱은 **진단·치료를 위한 의학 자료가 아니며, 자가진단용은 더더욱 아닙니다.**

- 수록된 풀이는 의료진 검수를 거쳤지만, **실무 이해를 돕기 위해 의도적으로 쉽게·간략하게** 정리한 설명입니다.
- 따라서 **최신 진단 기준·치료 지침과 다를 수 있고, 개별 환자의 상태를 설명하지 않습니다.**
- 실제 진단·치료·임상적 판단은 반드시 담당 의료진과 공식 진료기록·최신 의학 자료를 따라야 합니다.
- 보험·심사·청구 등에서의 최종 판단 역시 각 기관의 공식 기준과 원자료를 근거로 해야 하며, 본 앱의 설명을 근거로 삼지 마십시오.

요약하면, 이 앱은 **"코드의 대략적 의미를 빠르게 떠올리기 위한 보조 메모"** 수준으로 활용하시기 바랍니다.

## 누구를 위한 도구인가

환자가 아니라, 업무상 진단 코드를 자주 접하지만 일일이 의학 교과서를 찾기는 번거로운 **현장 실무자**를 위해 만들었습니다.

- **진료보조인력** — 차트·기록을 다루며 코드의 대략적 의미를 빠르게 확인해야 하는 분
- **원무과** — 접수·수납·증명서 발급 과정에서 진단 코드를 접하는 분
- **의무기록실** — 코드와 병명을 대조·정리하는 분
- **보험심사** — 청구·심사 과정에서 다수의 상병 코드를 검토하는 분
- **실손보험 판매·상담** — 고객 서류의 진단명을 쉽게 이해하고 설명해야 하는 분

의학용어가 또 다른 의학용어로 설명되어 생기는 "이해의 벽"을, 고등학생 수준의 쉬운 풀이와 일상 비유로 낮추는 것이 목표입니다.

## 한 화면에서 제공하는 것

- 정식 한국어 병명과 영문명
- 쉬운 풀이 + 일상 비유 (실무 이해용)
- 환자가 흔히 호소하는 증상
- 응급 신호 (해당 코드와 연관된 위험 징후 안내)
- 함께 자주 등장하는 관련 코드

## 주요 기능

- **통합 검색** — 코드(`J45`), 한국어 병명(천식), 별칭(고혈압) 모두 매칭되는 자동완성
- **분야별 보기** — 진료 분야 버튼을 누르면 해당 분야 코드만 표시 (초기 화면은 분야 선택형)
- **즐겨찾기** — 자주 보는 코드를 별표로 저장, 기기에 보관
- **최근 본 진단명** — 직전에 열어본 코드를 초기 화면에서 바로 다시 열기
- **응급 키워드 감지** — '응급·의식·호흡곤란' 등 입력 시 119 안내 우선 노출
- **검수 라벨** — 각 풀이가 의료진 검수를 거쳤음을 명시
- **면책 고지 시스템** — 본문·모달·푸터에 반복 노출
- **출처 명시** — KCD-8, WHO ICD-10, HIRA 통계 출처 표기
- **광고 없음** — 특정 병의원·의약품 추천 일체 없음

## 데이터 처리 구조 (3단 캐시)

호출 비용을 최소화하면서도 폭넓은 코드를 다루기 위한 3단 구조입니다.

| 단계 | 범위 | 처리 방식 |
|---|---|---|
| **Tier 1** — 내장 사전 | 외래 다빈도 100개 | 저장소에 JSON으로 내장, 의료진 검수, 즉시 응답, 호출 비용 0 |
| **Tier 2** — 서버 캐시 | 그 외 검색되는 코드 | 첫 사용자만 AI 호출 → 결과를 저장 → 이후 재사용 |
| **Tier 3** — 실시간 호출 | 희귀·신규 코드 | AI 직접 호출, 'AI 자동 생성' 라벨 명시 |

현재 공개 버전에는 **Tier 1만** 구현되어 있으며, 전체 코드 무제한 검색(Tier 2·3)은 구독 플랜에서 제공될 예정입니다.

## 콘텐츠 검수 워크플로우

진단명 풀이는 다음 단계로 관리합니다.

```
[1] AI로 초안 생성  →  [2] 의료진이 행 단위 검수·편집  →  [3] JSON 변환  →  [4] 앱 반영
```

검수용 템플릿은 의료진이 코드를 보지 않고 워드 편집과 동일한 방식으로 검토하도록 설계되었습니다.

## 면책 고지

본 앱이 제공하는 모든 정보는 **실무자의 이해를 돕기 위한 참고용 보조 자료**이며, 의학적 진단·치료를 대체할 수 없습니다.

- 수록 설명은 쉽게 간략화한 것으로, **최신 진단 기준·치료 지침과 다를 수 있습니다.**
- **환자 자가진단 용도로 사용하지 마십시오.** 증상이나 진단명에 대한 정확한 정보가 필요하면 반드시 의료진과 직접 상담해야 합니다.
- 응급 상황(흉통·호흡곤란·의식 저하·심한 출혈 등)에서는 검색이 아닌 **119 신고 또는 응급실 방문**이 우선입니다.
- 보험·심사·청구 등 업무상 판단은 각 기관의 공식 기준과 원자료를 근거로 해야 하며, 본 앱의 설명을 근거로 삼지 마십시오.
- 본 앱은 특정 의료기관·의약품을 추천하거나 광고하지 않습니다.
